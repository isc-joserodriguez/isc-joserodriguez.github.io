## Contexto

Proyecto desarrollado como práctica de maquetación web, replicando el diseño del landing page de Matcha, una plataforma de content marketing.

## Características

- Diseño responsive utilizando Bootstrap
- Navegación con scroll suave
- Secciones informativas sobre el producto
- Call-to-actions estratégicamente ubicados
- Optimizado para diferentes dispositivos

## Stack Técnico

Este proyecto utiliza tecnologías web fundamentales:

- **HTML5** para la estructura semántica
- **CSS3** para estilos personalizados
- **Bootstrap** como framework CSS para el diseño responsive
- **JavaScript** para interactividad básica

## Aprendizajes

- Maquetación con Bootstrap
- Diseño responsive mobile-first
- Buenas prácticas de HTML semántico
- Optimización de imágenes para web
