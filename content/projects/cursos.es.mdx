## Contexto

Plataforma educativa desarrollada para gestionar y tomar cursos en línea, permitiendo a los usuarios explorar diferentes categorías de cursos.

## Características

- Catálogo de cursos organizados por categorías
- Sistema de navegación intuitivo
- Interfaz limpia y enfocada en el contenido
- Diseño responsive para dispositivos móviles

## Stack Técnico

Desarrollado con tecnologías modernas de frontend:

- **React** para la interfaz de usuario
- **JavaScript (ES6+)** para la lógica de la aplicación
- **CSS3** para estilos personalizados
- **HTML5** para la estructura

## Aprendizajes

- Arquitectura de componentes en React
- Gestión de estado local
- Diseño de interfaces educativas
- Navegación y routing en aplicaciones SPA
