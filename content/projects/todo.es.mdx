## Contexto

Aplicación de lista de tareas construida completamente con JavaScript vanilla como ejercicio de programación sin dependencias de frameworks.

## Características

- Agregar nuevas tareas
- Marcar tareas como completadas
- Eliminar tareas
- Persistencia de datos en LocalStorage
- Interfaz minimalista y funcional
- **Desafío especial**: Todo el DOM es manipulado desde JavaScript, el HTML solo contiene un div root

## Stack Técnico

Desarrollado sin frameworks para demostrar conocimientos fundamentales:

- **JavaScript Vanilla** para toda la lógica y manipulación del DOM
- **HTML5** con estructura mínima (solo root div)
- **CSS3** para estilos
- **LocalStorage** para persistencia de datos

## Aprendizajes

- Manipulación avanzada del DOM
- Event handling en JavaScript puro
- Arquitectura de aplicaciones sin frameworks
- Gestión de estado sin librerías externas
- LocalStorage API
