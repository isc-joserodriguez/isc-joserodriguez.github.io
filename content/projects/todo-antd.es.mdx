## Contexto

Aplicación de lista de tareas desarrollada con React y Ant Design para demostrar el uso de una librería de componentes UI profesional.

## Características

- Interfaz profesional con componentes de Ant Design
- Agregar, editar y eliminar tareas
- Marcar tareas como completadas
- Filtros por estado (todas, activas, completadas)
- Contador de tareas pendientes
- Diseño limpio y moderno
- Responsive design

## Stack Técnico

Utilizando tecnologías modernas y librerías UI:

- **React** para la construcción de la interfaz
- **Ant Design** como librería de componentes UI
- **JavaScript (ES6+)** para la lógica
- **CSS3** para estilos personalizados

## Aprendizajes

- Integración de Ant Design con React
- Uso de componentes UI pre-construidos
- Gestión de estado en React
- Implementación de filtros y búsqueda
- Buenas prácticas con librerías de componentes
