## Contexto

Durante mi aprendizaje de React, decidí clonar la funcionalidad de búsqueda de Mercado Libre para practicar el consumo de APIs REST y manejo de estado. El objetivo era replicar la experiencia de usuario de búsqueda, paginación y visualización de productos.

## Retos

- **Consumo de API externa**: Integración con la API pública de Mercado Libre para obtener datos en tiempo real
- **Performance**: Optimización de búsquedas y renderizado de listas grandes de productos
- **UX**: Replicar la experiencia de usuario de Mercado Libre con diseño responsive

## Solución Técnica

Desarrollé una aplicación React que consume la API de Mercado Libre mediante Axios. Implementé:

- Componentes funcionales con Hooks (useState, useEffect)
- Debouncing en el campo de búsqueda para reducir llamadas a la API
- CSS Modules para estilos encapsulados y mantenibles
- Lazy loading de imágenes para mejorar performance
- Diseño responsive mobile-first

## Impacto

Este proyecto me permitió consolidar conocimientos en:
- Consumo de APIs REST y manejo de promesas
- Optimización de performance en React
- Diseño de interfaces responsive
- Gestión de estado en aplicaciones React
