## Contexto

Landing page diseñado para una cafetería ficticia especializada en café mexicano, con un diseño visualmente atractivo y moderno.

## Características

- Diseño visual atractivo con imágenes de alta calidad
- Secciones informativas sobre el café mexicano
- Call-to-action para contacto
- Galería de productos
- Diseño responsive
- Animaciones CSS suaves

## Stack Técnico

Construido con tecnologías web fundamentales:

- **HTML5** para estructura semántica
- **CSS3** con animaciones y transiciones
- **JavaScript** para interactividad

## Aprendizajes

- Diseño visual enfocado en el producto
- Uso efectivo de imágenes y tipografía
- Animaciones CSS para mejorar UX
- Principios de diseño para landing pages
- Storytelling visual
